В проекте испольузется база данных с названием ClientsDB. MS SQL 2016.

Базу данных в формате *.mdf приложил, она находится в этой же папке.

В БД есть 4 таблицы: 
1) AllNaturalClients - таблица физ лиц
2) AllLegalClients - таблица юр лиц
3) AllVipNaturalClients - таблица VIP физ лиц
4) AllVipLegalClients - таблица VIP юр лиц.

Коды (запросы) таблиц находятся в папке "Таблицы базы данных".

Инструкция по подключению:
1) Открыть обозреватель сервров (Вид -> Обозреватель серверов)
2) ПКМ на Подключение данных - > Добавить подключение
3) 
	Источник данных: Microsoft SQL Server
	Поставщик данных: Поставщик данных .NET Framework для SQL Server
4) Имя сервера: (localdb)\MSSQLLocalDB
5) Выберете или введите имя базы данных: ClientsDB -> OK
6) Разворачиваем базу данных в Обозревателе серверов.
7) Таблицы -> ПКМ -> Добавить новую таблицу
8) Создаем 4 таблицы с названиями, что указаны выше и внутри них прописываем строчки, которые расположены 
в *.txt файлах в папке "Таблицы базы данных".
9)  После создания таблиц - щелкаем на базу данных в обозревателе серверов. Смотрим на строку подключения,
которая расположена в окне свойств.
Сверяем строку подключения с данными в connect.xml, который расположен в папке: 
										"../дз17/HomeWork17/BankSystem/Connect"
По этому файлу происходит коннект к Базе данных.
10) Тк наша база данных пуста, то ее нужно заполнить. Заполнить ее можно как из прогарммы, там и напрямую заполнять таблицы 
в SQL Server. Если мы хотим заполнить таблицы быстро -> запускаем программу -> выбираем вход, как сотрудник -> 
Меню -> Рандомные клиенты -> Вводим число клиентов -> "Начать" -> Готово, можно пользоваться программой.

Информация по логированию находится в файле readme.txt, что расположен по пути:
"дз17/HomeWork17/BankSystem/BankSystem/Text/readme.txt".
