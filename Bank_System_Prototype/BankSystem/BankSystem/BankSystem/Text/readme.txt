==========================================================================КЛИЕНТ===================================================================
Чтобы войти клиенту в систему, ему нужно выбрать галочку "клиент", выбрать отдел в котором он числиться и ввести свой Id
(Id у каждого клиента свой, Id считается от 1 до n).
Пароль вводить не нужно, тк пока не реализовано его сокрытие.
Для клиентов из отделов Физический и Юридический по умолчанию закрыты функции Открытия вклада и кредита. Вклад и кредит им может открыть
только сотрудник банка. 
У клиентов из VIP отдела есть привелегиии и доверительное положение. Они могут открывать кредит и вклад без участия сотрудника банка. Но они так же 
будут записаны в логи.
ДЛЯ ВСЕХ КЛИЕНТОВ, НЕЗАВИСИМО ОТ ОТДЕЛА, ДОСТУПНЫ СЛЕДУЮЩИЕ ФУНКЦИИ:
1) ВХОД ПО ID (В БУДУЩЕМ ПО ПАРОЛЮ)
2) ТРАНЗАКЦИИ МЕЖДУ КЛИЕНТАМИ
3) ЗАКРЫТИЕ ВКЛАДА
4) ЗАКРЫТИЕ КРЕДИТА
